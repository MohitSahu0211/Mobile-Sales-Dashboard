# 📊 Mobile Sales Analytics Dashboard

## Overview
This project is a Power BI dashboard that analyzes mobile sales performance across cities, brands, and customer behavior.

## Key Insights
• Total Sales: 769M+
• Top City: Delhi
• Most Used Payment Method: UPI
• Highest Rated Sales contribute maximum revenue

## Features
• Interactive filters (Month, Brand, Payment Method)
• Sales distribution by cities (Map visualization)
• Customer rating analysis
• Yearly sales trend

## 🛠 Tools Used
• Power BI
• Excel / Dataset
• Data Visualization
